# GMP CheckMaster AI - 해커톤 프로젝트

## 프로젝트 개요
제약업계 GMP 규정 준수를 위한 지능형 체크리스트 관리 시스템

**핵심 기능:**
- AI 기반 건강상태 판정 (Amazon Q Developer + Bedrock)
- 지능형 패턴 분석 및 예측
- 실시간 출입 통제 연동
- 스마트 알림 시스템

- **팀원**: 백승재 (SAP 개발자), 풍기덕 (SAP 운영자)
