# 출입통제형 체크리스트 사용자 스토리

## 개요
AI 판정을 통해 출입 가능/불가를 결정하는 체크리스트 관련 사용자 스토리

## 작업자 스토리

### US-A01: 출입 전 건강상태 체크
**As a** 작업자  
**I want to** 모바일로 간편하게 건강상태를 체크하고  
**So that** 빠르게 출입 가능 여부를 확인할 수 있다

**상세 시나리오:**
- 출근 시 스마트폰으로 "출입 전 건강체크" 접속
- 발열, 설사, 체온, 기침 등을 간단히 입력
- 1-2분 내 완료 가능
- 즉시 출입 가능/불가 결과 확인

**수용 기준:**
- [ ] 모바일 친화적 UI
- [ ] 3분 이내 체크 완료
- [ ] 즉시 AI 판정 결과 표시
- [ ] 오프라인에서도 임시 저장 가능

**우선순위**: HIGH (MVP 필수)

### US-A02: AI 기반 출입 판정
**As a** 작업자  
**I want to** 체크 완료 즉시 AI가 출입 가능 여부를 판정해주길 원하고  
**So that** 관리자 확인을 기다리지 않고 바로 작업을 시작할 수 있다

**상세 시나리오:**
- 체크리스트 제출 즉시 AI 분석 시작
- "출입 가능", "출입 불가", "재확인 필요", "의료진 상담" 중 하나 결과
- 판정 근거를 쉬운 말로 설명
- 출입 불가 시 구체적 안내 제공

**수용 기준:**
- [ ] 5초 이내 판정 완료
- [ ] 판정 근거 자연어 설명
- [ ] 95% 이상 정확도
- [ ] 친근하고 명확한 메시지 제공

**우선순위**: HIGH (MVP 필수)

### US-A03: 출입 판정 결과 알림
**As a** 작업자  
**I want to** 출입 판정 결과를 명확하고 친근한 메시지로 받고 싶고  
**So that** 다음에 무엇을 해야 하는지 바로 알 수 있다

**상세 시나리오:**
- 출입 가능: "✅ 체크 완료! 안전하게 작업하세요"
- 출입 불가: "❌ 오늘은 출근이 어렵습니다. 충분히 휴식 후 내일 다시 체크해주세요"
- 재확인: "⚠️ 일부 항목 재확인이 필요합니다. 관리자에게 문의하세요"
- 의료상담: "🏥 의료진 상담 후 출입 가능 여부를 결정하겠습니다"

**수용 기준:**
- [ ] 앱 내 + 푸시 알림
- [ ] 상황별 맞춤 메시지
- [ ] 다음 행동 가이드 제공
- [ ] SMS 알림 (선택사항)

**우선순위**: HIGH (MVP 필수)

### US-A04: 출입 기록 조회
**As a** 작업자  
**I want to** 내 출입 체크 기록과 판정 결과를 조회하고  
**So that** 건강 상태 변화를 확인하고 패턴을 파악할 수 있다

**상세 시나리오:**
- 최근 7일간 체크 기록 조회
- 판정 결과 및 사유 확인
- 체온 변화 그래프 표시
- 출입 불가 이력 및 사유 확인

**수용 기준:**
- [ ] 직관적인 기록 조회 UI
- [ ] 날짜별 필터링 기능
- [ ] 시각적 차트 표시
- [ ] 상세 정보 드릴다운

**우선순위**: MEDIUM (확장 기능)

## 관리자 스토리

### US-A05: 출입통제 템플릿 생성
**As a** 관리자  
**I want to** 출입통제용 체크리스트 템플릿을 생성하고  
**So that** 우리 회사 기준에 맞는 건강 체크를 할 수 있다

**상세 시나리오:**
- "출입통제형" 유형 선택
- 건강 관련 질문 항목 추가 (발열, 설사, 체온, 기침 등)
- 각 항목별 위험도 가중치 설정
- AI 판정 규칙 설정 (자동 거부 조건 등)

**수용 기준:**
- [ ] 직관적인 템플릿 생성 UI
- [ ] 항목별 위험도 설정 기능
- [ ] 실시간 미리보기
- [ ] AI 판정 규칙 시각적 설정

**우선순위**: HIGH (MVP 필수)

### US-A06: AI 판정 규칙 설정
**As a** 관리자  
**I want to** 출입통제 체크리스트의 AI 판정 규칙을 세밀하게 설정하고  
**So that** 우리 회사 안전 기준에 맞는 정확한 판정이 이루어질 수 있다

**상세 시나리오:**
- 자동 거부 조건 설정 (발열=있음, 설사=있음, 체온>37.5도)
- 경고 조건 설정 (기침=있음, 컨디션=나쁨)
- 항목별 위험도 가중치 조정 (1-10점)
- 종합 점수 기준 설정 (80점 이상 통과)

**수용 기준:**
- [ ] 조건별 규칙 설정 UI
- [ ] 위험도 가중치 슬라이더
- [ ] 규칙 테스트 기능
- [ ] 규칙 변경 이력 관리

**우선순위**: MEDIUM (확장 기능)

## 책임자 스토리

### US-A07: 출입 현황 실시간 모니터링
**As a** 책임자  
**I want to** 우리 팀원들의 출입 체크 현황을 실시간으로 모니터링하고  
**So that** 누가 아직 체크하지 않았거나 출입 불가 판정을 받았는지 바로 알 수 있다

**상세 시나리오:**
- 팀별 출입 체크 완료율 대시보드 (12/15명 완료)
- 미완료자 실시간 표시 (빨간색 표시)
- 출입 불가 판정자 즉시 알림 (주황색 표시)
- 전체 완료 시 완료 알림

**수용 기준:**
- [ ] 실시간 현황 업데이트 (30초 이내)
- [ ] 직관적인 대시보드 UI
- [ ] 색상으로 상태 구분
- [ ] 모바일에서 현황 확인 가능

**우선순위**: HIGH (MVP 필수)

### US-A08: 출입 불가자 즉시 알림
**As a** 책임자  
**I want to** 팀원 중 출입 불가 판정을 받은 사람이 있으면 즉시 알림을 받고  
**So that** 빠르게 대응하고 업무 계획을 조정할 수 있다

**상세 시나리오:**
- 출입 불가 판정 시 즉시 푸시 알림
- 판정 사유 및 해당 직원 정보 표시
- 대체 인력 필요 여부 안내
- 의료진 상담 필요 시 별도 알림

**수용 기준:**
- [ ] 즉시 알림 (1분 이내)
- [ ] 상세한 상황 정보 제공
- [ ] 다중 알림 채널 (푸시/SMS/이메일)
- [ ] 알림 히스토리 관리

**우선순위**: MEDIUM (확장 기능)

## 판정 결과별 메시지 예시

### 출입 가능
- "✅ 체크 완료! 안전하게 작업하세요"
- "모든 항목이 정상입니다. 좋은 하루 되세요!"
- "건강상태 양호! 오늘도 화이팅하세요 💪"

### 출입 불가
- "❌ 오늘은 출근이 어렵습니다. 충분히 휴식 후 내일 다시 체크해주세요"
- "건강을 위해 오늘은 집에서 쉬시기 바랍니다 🏠"
- "발열 증상이 있습니다. 해열제 복용 후 경과를 지켜보세요"

### 재확인 필요
- "⚠️ 일부 항목 재확인이 필요합니다. 관리자에게 문의하세요"
- "추가 확인이 필요합니다. 잠시 후 다시 체크해주세요"
- "체온을 다시 측정해주세요. 정확한 측정이 중요합니다 🌡️"

### 의료진 상담
- "🏥 의료진 상담 후 출입 가능 여부를 결정하겠습니다"
- "보건실에서 상담 받으신 후 다시 체크해주세요"
- "증상이 지속되면 병원 진료를 받아보시기 바랍니다"
