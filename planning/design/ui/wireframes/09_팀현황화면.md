# 09. 팀 현황 화면 와이어프레임 (책임자용)

## 화면 개요
- **목적**: 담당 팀 완료 현황 모니터링 + 승인 관리 + 팀원 소통
- **사용자**: 책임자 (이책임)
- **플랫폼**: 모바일 우선 (언제 어디서나 팀 현황 확인)

## 상호작용 설계

### 메뉴 네비게이션
1. **[현황]** → 팀 완료 현황 (기본 화면)
2. **[승인]** → 부적합자 조치 승인 관리
3. **[소통]** → 팀원 공지사항 + 채팅
4. **[설정]** → 알림 설정, 개인 정보

### 실시간 모니터링
1. **30초 간격 업데이트** → 완료율, 팀원 상태 실시간 갱신
2. **푸시 알림** → 승인 요청, 긴급 상황 즉시 알림
3. **색상 변화** → 마감 임박 시 빨간색 강조
4. **진동 알림** → 중요 알림 시 진동 피드백

### 승인 프로세스
1. **승인 요청 수신** → 운영자로부터 승인 요청
2. **상세 내용 검토** → 조치 사유, 내용, 일정 확인
3. **승인/거부 결정** → 원터치로 즉시 처리
4. **자동 알림 발송** → 결정 사항 운영자에게 즉시 전달

### 팀원 소통
1. **공지사항 작성** → 팀 전체에게 중요 공지
2. **실시간 채팅** → 팀원과 즉시 소통
3. **읽음 확인** → 공지사항 읽음 여부 추적
4. **일괄 연락** → 미완료자에게 동시 전화/메시지
