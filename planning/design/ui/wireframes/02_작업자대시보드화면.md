# 02. 작업자 대시보드 화면 와이어프레임

## 화면 개요
- **목적**: 작업자의 메인 화면 - 오늘 할 일, 주간 현황, 나의 기록 통합 관리
- **사용자**: 작업자 (김작업, 박작업)
- **플랫폼**: 모바일 우선 (세로 스크롤 구조)

## 상호작용 설계

### 메뉴 네비게이션
1. **빠른 메뉴**
   - [오늘] 클릭 → "📅 오늘 할 일" 섹션으로 스크롤
   - [이번주] 클릭 → "📊 이번 주 현황" 섹션으로 스크롤  
   - [기록] 클릭 → "📈 나의 기록" 섹션으로 스크롤
   - [액션] 클릭 → "🚀 빠른 액션" 섹션으로 스크롤
   - 부드러운 스크롤 애니메이션 (0.5초)

2. **섹션 앵커**
   - 각 섹션에 ⚓ 앵커 표시
   - 현재 보고 있는 섹션 메뉴 하이라이트
   - 스크롤 위치에 따른 메뉴 활성화
1. **체크리스트 작성하기**
   - 클릭 → 03_체크리스트화면으로 이동
   - 미완료 시: 큰 버튼으로 강조
   - 완료 시: "결과보기" 버튼으로 변경

2. **결과보기**
   - 완료 후에만 활성화
   - 클릭 → 04_결과화면으로 이동

### 정보 확인
1. **주간 현황**
   - 각 날짜 클릭 → 해당 날짜 상세 기록
   - 완료율 클릭 → 주간 상세 통계

2. **다음 주 미리보기**
   - 각 일정 클릭 → 상세 정보
   - 특별 일정 알림 표시

3. **나의 기록**
   - 달력 날짜 클릭 → 해당 날짜 기록 상세
   - 완료된 날짜: 체크리스트 결과 표시
   - 예정된 날짜: 스케줄 정보 표시
   - 성과 항목 클릭 → 상세 통계

### 빠른 액션
1. **운영팀 문의**
   - 클릭 → 전화 앱 실행 (tel:1234)
   - 또는 메신저 앱 연동

2. **전체 일정보기**
   - 클릭 → 월간 달력 뷰
   - 필터링: 완료/미완료/예정

## 데이터 구조

```javascript
// 스크롤 네비게이션 함수 (새로 추가)
function scrollToSection(sectionId) {
  const element = document.getElementById(sectionId + '-section');
  element.scrollIntoView({ 
    behavior: 'smooth',
    block: 'start'
  });
  
  // 메뉴 활성화 상태 업데이트
  updateActiveMenu(sectionId);
}

function updateActiveMenu(sectionId) {
  document.querySelectorAll('.menu-item').forEach(item => {
    item.classList.remove('active');
  });
  document.querySelector(`[data-section="${sectionId}"]`).classList.add('active');
}

// 스크롤 위치에 따른 메뉴 자동 활성화
window.addEventListener('scroll', () => {
  const sections = ['today', 'weekly', 'records', 'actions'];
  const scrollPos = window.scrollY + 150;
  
  sections.forEach(section => {
    const element = document.getElementById(section + '-section');
    if (element.offsetTop <= scrollPos && 
        element.offsetTop + element.offsetHeight > scrollPos) {
      updateActiveMenu(section);
    }
  });
});
```

## 접근성 고려사항
- **키보드 네비게이션**: Tab으로 모든 버튼 접근, 달력 날짜 키보드 탐색
- **스크린 리더**: 상태 변화 시 음성 안내, 달력 날짜별 상태 읽기
- **색상 대비**: WCAG 2.1 AA 기준 준수
- **터치 영역**: 모든 버튼 44px 이상, 달력 셀 32px (터치 가능)
- **상태 표시**: 색상 + 아이콘 + 텍스트 조합으로 달력 상태 구분

## 성능 최적화
- **지연 로딩**: 스크롤 시 섹션별 로딩
- **캐싱**: 주간/월간 데이터 로컬 캐시
- **실시간 업데이트**: WebSocket으로 상태 변화 감지
- **오프라인 지원**: 기본 정보 오프라인 표시
